Cântico Selecionado

Letra do cântico selecionado aparecerá aqui. Este é um espaço para exibir o conteúdo completo do cântico, permitindo que os usuários o leiam e acompanhem durante a missa. O texto pode ser longo e, por isso, é importante que este quadro tenha barras de rolagem para facilitar a visualização de todo o conteúdo.
